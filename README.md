# Dotfiles

## Requirements 

### Git
### Stow 

## Setup

create dotfiles dir if not present 

```
$ git clone git@github.com:swapnilsaahu/dotfiles.git
$ cd dotfiles
```
create symlinks using stow
```
$ stow .
```
to avoid conflicts in stow
```
$ stow --adopt .
```
moves the conflicting file/dir inside dotfiles
